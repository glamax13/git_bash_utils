usage: parser [-a] [-e extension] [-n needle] [-f from] [-t threads] [-l logs]

-a for advanced
	Doens't only find the file but also hte line(s) with a match

-e for extension
	Only search into files with this specific extension

-n for needle
	The string to find in the files

-f for from
	The folder to search

-t for threads
	The amount of concurrent jobs to run simultaneously

-l for logs
	The path where the logs must be written
